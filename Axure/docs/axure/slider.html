To set the initial value of the slider, change only the value in the text input, the slider thumb will be positioned with the Loaded event of this text input.

Slider can be displayed as disabled with “not allowed” cursor. A hot spot has the cursor property, and it  is shown and brought forward with its Loaded event if the component is disabled in the design.
