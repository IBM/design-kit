About variables

Variables in Axure RP are structures used to store alphanumeric information (the variable's "value"). Variable values can then be used in a variety of ways—displayed or manipulated using actions. Variables must be named, and can optionally be given default starting values.

Variables in this library

Elements of a widget library need to work unrelated to any document, therefore local variables are used. They are "built-in" to the widgets, mostly using a dedicated text element when storing a value is required to complete various actions.

Editing components

Any element that has a name that starts with "var" (e.g. varSelected") needs to remain part of the component to maintain the designed behavior.

Read more on axure.com

Role of variables in components:

Count selected items (dropdown, data table)

–

Store variable height of a widget to control moving or resizing other widgets (content switcher, multi line code snippet etc.)

–

Dynamically control positioning of a widget based on a selected item in a repeater data set (select)

–

Control default selection in repeater-based component (select)
