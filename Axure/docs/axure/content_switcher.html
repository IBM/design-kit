Content is switched (dynamic panel).

Background is resized, below widgets are pushed/pulled when widget size is changing due to different content height in sections.

Implemented features

Content can be edited on states of the "pnl content" dynamic panel.

Content (panel state) is switched with the Selected event of a button shape.

A "background" element is included with the widget, this is the target of the resize actions. If you need a different background, don't replace it, just change styles of the included element.
